Help

Create a Plot

To create a plot, simply click the "Create Plot" icon, input a name, and click "Create".

Delete a Plot

To delete a plot, click the red "Delete Plot" button, then click the "Delete" button next to the plot you wish to delete.

Search Produce

To search for produce, click the magnifying glass icon in the upper right corner, then type a produce name, such as "Corn". If that produce is located in a plot, it will show up below the input box. If you wish to go to the plot where the produce is located, simply click on it.

Add a Garden Row

Once you're in a plot, click the "Add Row" button and a window will appear. After you have filled out all the row's data in the window, click the "Add" button.

Update/Delete a Garden Row

You can update or delete a row by double clicking on it and choosing your option.
